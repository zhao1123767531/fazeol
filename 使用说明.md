# 法泽在线动态版

启动方式：

```bash
npm start
```

打开：

```text
http://localhost:3000
```

现在账号、课程、考试、课表、提交、批改、消息等会保存到 `data/state.json`。账号密码不会再由前端明文校验，而是在后端加密保存和验证。

真实上传文件会保存到 `uploads/`：

- 教师在课程课时里上传视频 / PDF 后，学生可以在线播放视频或下载资料。
- 教师上传试卷文件后，学生下载的是教师上传的原始文件。
- 学生按题上传答卷时，文件也会真实保存，并在提交记录里保留下载地址。

如果直接双击打开 `index.html`，页面仍可演示，但只会使用浏览器本地数据。

## 部署到 Vercel 的说明

当前版本适合在本地服务器、云服务器、Render、Railway 等可以保留磁盘文件的环境运行。

当前版本已经加入云服务适配。配置环境变量后自动启用：

```text
DATABASE_URL=postgres://...
BLOB_READ_WRITE_TOKEN=vercel_blob_rw_...
```

- `DATABASE_URL` 用于保存账号、课程、考试、成绩等结构化数据。
- `BLOB_READ_WRITE_TOKEN` 用于保存视频、PDF、试卷和答卷文件。

Vercel Functions 对直接上传的大文件有限制；正式大视频场景建议下一步升级为 Vercel Blob 客户端直传。
