# 法泽在线动态版

启动方式：

```bash
npm start
```

打开：

```text
http://localhost:3000
```

现在账号、课程、考试、课表、提交、批改、消息等会保存到 `data/state.json`。账号密码不会再由前端明文校验，而是在后端加密保存和验证。

真实上传文件会保存到 `uploads/`：

- 教师在课程课时里上传视频 / PDF 后，学生可以在线播放视频或下载资料。
- 教师上传试卷文件后，学生下载的是教师上传的原始文件。
- 学生按题上传答卷时，文件也会真实保存，并在提交记录里保留下载地址。

如果直接双击打开 `index.html`，页面仍可演示，但只会使用浏览器本地数据。

## 部署到 Vercel 的说明

当前版本适合在本地服务器、云服务器、Render、Railway 等可以保留磁盘文件的环境运行。

如果部署到 Vercel，需要把两类存储换成云服务：

- 账号、课程、考试、成绩等结构化数据：Vercel Postgres、Neon 或 Supabase。
- 视频、PDF、答卷文件：Vercel Blob、S3、Cloudflare R2 或 Supabase Storage。

原因是 Vercel 的 Serverless 运行环境不会长期保留 `data/` 和 `uploads/` 目录里的文件。代码里的接口已经按“后端 API”方式整理好了，下一步接 Vercel 数据库和 Blob 会比较顺。
